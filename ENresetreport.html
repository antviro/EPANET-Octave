Function File: errorcode = ENresetreport ( )

Clears any report formatting commands that either appeared in the [REPORT] section of the EPANET Input file or were issued with the ENsetreport function. After calling this function the default reporting options are in effect. These are: No status report No energy report No nodes reported on No links reported on Node variables reported to 2 decimal places Link variables reported to 2 decimal places (3 for friction factor) Node variables reported are elevation, head, pressure, and quality Link variables reported are flow, velocity, and head loss It returns an error code.

See also: ENsaveH.
