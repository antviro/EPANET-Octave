Function File: errorcode = ENepanet (fileinp, filereport, filebin)

Epanets the Epanet Toolkit (with ENepanet.c). fileinp EPANET input file (inp). Compulsory argument. filereport output report file (in case used), if not given inp file name is used changing its extension with rpt instead filebin binary output file (in case used), if not given inp file name is used changing its extension with bin instead.
