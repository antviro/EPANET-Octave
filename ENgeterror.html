Function File: errormessage,newerrorcode = ENgeterror(errorcode)

Runs a step of hydraulic simulation. errormessage is the corresponding message(s) of error Outputs: errorcode is the number of the error we want to know It can be run vectorized where errormessage would be a string cell array newerrorcode can also be retrieved

See also: ENgeterror, ENgetnodevalue, ENsetnodevalue.
