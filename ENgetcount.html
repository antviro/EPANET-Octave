Function File: number,errorcode = ENgetcount(countcode)

Counts the number of network components of a specified type countcode is a string or a number with the specified type It can be run vectorized introducing countcode as a vector (or matrix) of numbers Outputs: number is the number we wanted to obtain errorcode can also be retrieved EN_NODECOUNT 0 EN_TANKCOUNT 1 reservoirs and tank nodes EN_LINKCOUNT 2 EN_PATCOUNT 3 time patterns EN_CURVECOUNT 4 curves EN_CONTROLCOUNT 5
