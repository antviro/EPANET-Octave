Function File: id,errorcode = ENgetpatternid(index)

Gets the text id of a (or some) patterns index is an integer which is the index of the pattern Outputs: id is a string with the pattern identifier, it should be a string variable containing the string and not the string itself. No idea of why it cause problem the later... It can be run vectorized where id would be a string cell array, in case index is an array errorcode can also be retrieved
