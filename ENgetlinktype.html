Function File: typecode,errorcode,stype = ENgetlinkvalue(id)

Gets the text id of a link (or several links) Outputs: typecode is the code of the type of link errorcode can also be retrieved stype is a string describing type of link Inputs: id is the integer index of the link (can be a vector) or empty [] and then all links are retrieved: ENgetlinktype([])

See also: ENgetlinkid, ENgetlinkvalue, ENsetlinkvalue.
